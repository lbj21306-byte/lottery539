# lottery539

註冊 GitHub
「New repository」

Repository name：lottery539
勾選 「Public」（必須是公開才能用 Pages）
勾選 「Add a README file」
按 「Create repository」
=========

第三步：上傳你的 HTML 檔案
進入剛建立的 repository，點 「Add file」 → 「Upload files」
把你下載的 lottery539.html 拖進去 → 按 「Commit changes」
⚠️ 重要： 把檔案改名為 index.html 再上傳（這樣網址不用加檔名）

第四步：開啟 GitHub Pages
在 repository 頁面，點上方 「Settings」 → 左側選單找 「Pages」
在 「Branch」 下拉選 「main」 → 按 「Save」
等約 1–2 分鐘

第五步：取得你的網址
重新整理 Pages 設定頁，會看到：

✅ Your site is live at https://你的Username.github.io/lottery539

這個網址就可以直接貼到 LINE 了！
